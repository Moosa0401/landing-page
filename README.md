FIFA World Cup 2026 Landing Page

A modern landing page inspired by The Odin Project’s Landing Page assignment, redesigned with a FIFA World Cup 2026 theme.

Overview

This project showcases information about the FIFA World Cup, including:

* A hero section introducing the tournament
* A featured section highlighting the top scorers in World Cup history
* An inspirational football quote inspired by Pelé
* A call-to-action section encouraging users to follow the tournament
* Interactive hover effects for images, buttons, and navigation links

Features

* Responsive Flexbox layout
* Custom World Cup theme
* Hover animations and transitions
* Clean and organized HTML structure
* Modern CSS styling
* Semantic and accessible markup

Technologies Used

* HTML5
* CSS3
* Flexbox
* Git & GitHub

What I Learned

Through this project, I practiced:

* Building complete webpage layouts using Flexbox
* Structuring content with HTML
* Styling components with CSS
* Creating hover effects and transitions
* Managing projects with Git and GitHub
* Organizing CSS into reusable sections
